Install text-magic with npm

```bash

  npm start
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/krushang-07/Text-magic.git
```

Go to the project directory

```bash
  cd text-magic
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**Client:** Reactjs, JavaScript
